## üö® Critical UX Issues Found

### Issue #1: NO Visual Distinction for Imposters
**Observation:** In the clue list, I can see:
- Player_3: "Guessing: beach" (IMPOSTER)
- Player_6: "Guessing: Sand" (IMPOSTER)
- Player_1: "Guessing: beach" (IMPOSTER)

**Problem:** Only indicated by small text "Guessing: X"
- Not prominent enough
- Easy to miss
- No color coding
- No icons

**Impact:** Hard to track who imposters are during game review

---

### Issue #2: Clues Blend Together
**Observation:** 12 clues shown as similar-looking cards

**Problems:**
- All look the same
- No visual hierarchy
- Imposter clues don't stand out
- Hard to scan quickly

**What's Missing:**
- Color coding (imposters = different color)
- Icons (üé≠ for imposter clues)
- Visual separation between rounds
- Highlighting of suspicious clues

---

### Issue #3: Inner Monologue Lacks Context
**Observation:** Thoughts show confidence % and text

**Problems:**
- Can't quickly see which PLAYER is thinking
- No avatar/visual connection to player circle
- Confidence % not visually represented (just text)
- All thoughts look identical

**What's Needed:**
- Player avatar next to thought
- Confidence bar (visual progress bar)
- Color-coding by player
- Connection to player in circle (highlight)

---

### Issue #4: Player Circle Too Static
**Observation:** Players shown as simple circles

**Problems:**
- No indication of who's currently thinking/acting
- All players look identical until game ends
- No animation or life
- Doesn't draw attention to active player

**What's Missing:**
- Pulsing glow on active speaker
- Larger size for current player
- Animation when giving clue
- Visual feedback during voting

---

### Issue #5: No Imposter Reveal Showing
**Observation:** Game completed (Round 2/2) but no results visible

**Problems:**
- SSE connection error in console
- ImposterReveal component not appearing
- Likely stream disconnected before game_complete event
- No dramatic reveal moment

**Root Cause:** Stream reliability issue

---

### Issue #6: Clue Display Too Small
**Observation:** Clue words in quotes, same size as metadata

**Problems:**
- "Granular" same size as "Player_6" label
- Should be 3x larger
- Doesn't emphasize the actual clue
- Hard to read clue at a glance

**What's Needed:**
```css
.clue-text {
  font-size: 2.5rem;  /* 3x larger */
  font-weight: 700;
  text-align: center;
  margin: 1rem 0;
}
```
